# Play environment for Py & R on Windows

## Setup

Install

- WSL (for WSL bash)
- Python 3.7
- R
- Rtools
- RStudio
- VSCode
- VSCode 'settings sync' extension + trigger sync

Run
`pip install wheel`
`pip install "https://github.com/matangover/mypyls/archive/master.zip#egg=mypyls[patched-mypy]"`
`./ruw install`
`./ruw lint`
`./ruw dcf`
